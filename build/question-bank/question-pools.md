# Question Pools

A question pool is a named group of related questions with a **select count** — the number of questions the assessment draws from the pool. Each student receives a different random selection, so no two students see exactly the same set.

For example, a pool of 20 equivalent questions with a select count of 5 means every student gets 5 of the 20, chosen at random.

## Why use a pool

* **Academic integrity** — students get different questions, so sharing or copying answers is far less useful. This complements [dynamic questions](../question-builder/dynamic-questions/README.md), which vary each question; pools vary *which* questions a student sees.
* **Reusable question banks** — maintain a large set of equivalent questions and sample from it across attempts, sections, or terms.
