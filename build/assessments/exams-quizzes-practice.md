# Exams, Quizzes & Practice

Exams, quizzes, and practice assessments are all built the same way — by adding questions from your [question bank](../question-bank/README.md). They differ in intent and settings (for example, how they're scheduled or weighted), not in how you assemble them.

## Build the assessment

{% stepper %}
{% step %}
**Add questions**

Click the box next to a question title in your question bank to add it to the assessment.
{% endstep %}

{% step %}
**Verify the questions are included**

Questions you add appear in the assessment window on the right-hand side. To remove one, uncheck the box next to its title.

{% hint style="info" %}
The number of questions, question pools, and points update automatically as you add questions.
{% endhint %}
{% endstep %}

{% step %}
**Set the question order**

By default, questions are grouped by type into **Question Groups** and randomized within each group. You can:

1. **Create new groups** — click the **+** button after clicking a question group title.
2. **Fix question order** — click the crossed-arrow icon.
3. **Combine groups** — drag a question into a different question group.

{% hint style="info" %}
Question Group titles are shown to students. One tip: combine all questions into a single group and name it the assessment title (e.g., "Final Exam"). In other cases you may want to fix long workout problems as the last questions.
{% endhint %}
{% endstep %}
{% endstepper %}
