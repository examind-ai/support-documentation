# Deliver

Delivering an assessment covers everything between building it and grading it: **when** students can take it, **monitoring** them as they do, **handling** the unexpected, and **protecting** the integrity of the results. This section also covers Excel Assignments, which are delivered to students through the EXAMIND Excel add-in.

## In this section

* [**Schedule**](schedule.md) — set availability windows, time limits, attempts, and per-student extensions.
* [**Live View**](live-view.md) — monitor attempts in real time and intervene (pause, add time, force submit) as needed.
* [**Offline Behavior**](offline-behavior.md) — how EXAMIND handles a student losing connectivity mid-assessment.
* [**Tactical Deterrents**](tactical-deterrents.md) — evidence-based measures that discourage misconduct during an assessment.
* [**Excel Assignments**](excel-assignments/README.md) — deliver spreadsheet work in Microsoft Excel through the EXAMIND add-in.
