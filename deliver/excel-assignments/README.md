# Excel Assignments

Excel Assignments let students complete spreadsheet work — models, calculations, and analyses — directly in Microsoft Excel, with each student receiving unique data and their work auto-graded. They run through the EXAMIND Excel add-in.

## How it works

* **Unique data per student.** Each student receives their own variables and datasets, so no two submissions are alike — academic integrity by design.
* **Automatic grading.** Submissions are graded against the assignment's answer logic, like other EXAMIND question types.
* **Real Excel, not a simulation.** Students work in Microsoft Excel through the EXAMIND add-in.
