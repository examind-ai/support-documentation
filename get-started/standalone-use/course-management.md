# Course Management

## Accessing Courses

1. Click **Courses** in the left menu

## Create New Course

1. Click **+ ADD COURSE**

## Course Settings

Input the Course Name, Course Code, and Start and End Dates.&#x20;

{% hint style="info" %}
Courses are listed as either active or past courses based on the start and end dates.&#x20;
{% endhint %}
